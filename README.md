# the Apate Project
A browser-based realtime image process application.

## ⚠️ Warning
This is an **experimental** project and is still in development. BUGs may exist. APIs and features may change.

## Technologies
This project is based on those technologies:
* Rust
* WebAssembly
* wasm-bindgen
* TypeScript
* React
* Mobx
